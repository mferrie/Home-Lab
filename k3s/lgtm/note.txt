To update Loki: helm repo add grafana-community https://grafana-community.github.io/helm-charts && helm repo update && helm install loki grafana-community/loki -n lgtm -f loki/values.yaml

To update Tempo: helm repo add grafana-community https://grafana-community.github.io/helm-charts && helm repo update && helm install tempo grafana-community/tempo -n lgtm -f tempo/values.yaml

To update Prometheus: helm repo add prometheus-community https://prometheus-community.github.io/helm-charts && helm repo update && helm install prometheus-community/prometheus -n lgtm -f values.yaml
